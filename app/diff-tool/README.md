# 文本对比工具

文本对比工具基于 Monaco DiffEditor 在浏览器本地比较原始文本和修改后文本，适合查看 JSON、Markdown、TXT、日志和常见代码文件的差异。你可以手动编辑，也可以上传 UTF-8 文本文件，工具会根据文件扩展名自动识别语法高亮。

## 功能说明

- 支持并排视图和逐行视图切换，直接使用 Monaco DiffEditor 渲染差异。
- 支持上传原始文本和修改后文本，单个文件限制 5MB，仅支持 UTF-8 编码。
- 支持 JSON、Markdown、TXT、HTML、CSS、JavaScript、TypeScript、YAML、XML、SQL、Shell、Python、Java、Go、Rust、C/C++ 等常见代码高亮。
- 支持手动切换两侧语法高亮，并通过 Monaco API 更新编辑器模型语言。
- 支持调用 Monaco 格式化能力，对两侧内容一键格式化；不支持格式化的语言会保持原内容。
- 支持清空、复制、下载、交换两侧内容、恢复示例、忽略行尾空白、自动换行和只读模式。
- 支持显示差异数量、新增行和删除行，并可跳转上一个或下一个差异。

## 使用说明

1. 在“原始文本”和“修改后文本”区域上传文件，或直接在 DiffEditor 中编辑内容。
2. 上传文件后会覆盖对应区域的当前内容，并按文件扩展名自动切换语法高亮。
3. 使用“视图”下拉框在并排和逐行对比之间切换。
4. 点击“格式化两侧”可调用 Monaco 的文档格式化能力整理左右文本。
5. 使用“上一个差异”和“下一个差异”在变更块之间快速定位。
6. 需要保留结果时，可以分别复制或下载原始文本和修改后文本。

## 安全说明

所有文本读取、编辑、格式化和差异计算都在当前浏览器中完成。上传文件不会发送到服务器，也不会被工具持久化存储，适合对配置、日志、接口响应和代码片段做临时对比。

## 限制说明

- 单个上传文件最大 5MB，过大的日志或构建产物建议先在本地截取关键片段。
- 仅支持 UTF-8 文本文件，其他编码可能会被拒绝或显示为乱码。
- 语法高亮和格式化能力取决于当前 Monaco 已注册的语言和格式化 Provider。
- Markdown、TXT、日志等纯文本会直接进入 diff，不做额外解析或自动格式化。

## 常见问题

**为什么点击格式化后没有变化？**

当前语言可能没有可用的 Monaco 格式化 Provider。JSON、HTML、CSS、JavaScript 和 TypeScript 通常可以格式化，纯文本和部分小众语言可能不会变化。

**上传文件会不会离开本机？**

不会。文件通过浏览器 File API 读取，内容只进入当前页面的 Monaco DiffEditor。

**为什么空格差异消失了？**

如果开启“忽略行尾空白”，DiffEditor 会忽略行尾空格和缩进末尾差异。关闭该开关即可查看空白变化。
