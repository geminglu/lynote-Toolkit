# JSON 格式化

这个工具用于校验、格式化、压缩、转义和查看 JSON 内容，适合处理原始 JSON、转义后的 JSON 字符串，以及普通文本转 JSON 字符串的场景。

1. 普通 JSON

```json
{"name":"tool","enabled":true}
```

2. 已转义的 JSON 字符串

```text
"{\"name\":\"tool\",\"enabled\":true}"
```

3. 外层带引号的 JSON 文本

```text
'{"appId":"wx6b3b6451b8e1fc15","targetHost":"https://xxx.sss.com.cn"}'
```

4. 普通字符串

```text
hello world
```

输入普通字符串时，右侧会立即转换为合法的 JSON 字符串格式，例如：

```json
"hello world"
```

## 使用说明

1. 在左侧输入原始内容，支持 JSON、JSON 字符串和普通文本。
2. 当内容可以被识别和解析时，右侧会自动展示格式化结果。
3. 如果左侧输入非法 JSON，页面顶部会显示错误提示，右侧不会更新。
4. 你可以使用工具栏对左右两侧分别执行格式化、压缩、转义、复制和下载操作。
5. 上传文件时，内容会默认导入到左侧主输入区。
6. 新建草稿后不会立即进入历史记录，只有开始编辑后才会自动保存。
